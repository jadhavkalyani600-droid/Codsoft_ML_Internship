# Handwritten Text Generation (RNN) ✍️

## 📌 Project Overview
An advanced project where I implemented a **Recurrent Neural Network (RNN)** from scratch to learn patterns in text and generate new strings that mimic handwritten sequences.



## 🛠️ Tech Stack
- **Model:** Character-level RNN.
- **Library:** NumPy (Pure Mathematical Implementation).
- **Language:** Python 3.14.

## 🤖 Why NumPy from Scratch?
Instead of using high-level libraries like TensorFlow, I built this model using NumPy to demonstrate a deep understanding of:
- **Backpropagation Through Time (BPTT).**
- **Hidden State management.**
- **Weight optimization.**

## 📁 Files in this Task
- `handwritten_gen_task.py`: RNN script.
- `GENERATED_TEXT_REPORT.txt`: The text generated by my model.

## 🚀 How to Run
Simply execute: `python handwritten_gen_task.py`
