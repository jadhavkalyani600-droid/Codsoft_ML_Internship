# Movie Genre Classification using Machine Learning

## 📌 Project Overview
This project utilizes Machine Learning to predict a movie's 'Genre' based on its plot descriptions (storylines). For example, if a story mentions 'detective' and 'murder', the model predicts the genre as 'Thriller'.



## 📊 Dataset
I have used the Movie Genre Dataset from Kaggle for this project.
- **Training Data:** 54,214 movies (Title, Genre, Description)
- **Test Data:** 54,200 movies (Title, Description)

## 🛠️ Tech Stack & Libraries
- **Language:** Python 3.14
- **Editor:** VS Code
- **Libraries:** - `Pandas` (For data handling)
  - `Scikit-learn` (For ML Algorithms and TF-IDF)
  - `Matplotlib` & `Seaborn` (For graph visualization)

## 🤖 Machine Learning Approach
I have used the **Logistic Regression** algorithm in this project.
1. **TF-IDF Vectorizer:** Used to convert text data into numerical format.
2. **Model Training:** The model was trained using the `train_data`.
3. **Evaluation:** The model achieved an accuracy of 57%, which is an excellent start for text-based classification.



## 📁 Project Structure
- `check_final.py`: Main Python script to run the model.
- `train_data.txt`: Dataset file.
- `MAZA_REPORT.csv`: Final predictions generated by the model.
- `README.md`: Project documentation.

## 🚀 How to Run
1. Run `pip install pandas scikit-learn matplotlib seaborn` in your VS Code terminal.
2. Execute the command `python check_final.py`.
3. The output will be displayed in the terminal and `MAZA_REPORT.csv` will be created.
