# CodSoft Machine Learning Internship 🚀

This repository contains the projects completed during my Machine Learning Internship at **CodSoft**. The internship focused on building real-world ML models using Python and various data science libraries.

## 📁 Project Overview

| Task # | Project Name | Description |
| :--- | :--- | :--- |
| **Task 1** | [Movie Genre Classification](./Task_1_Movie_Genre_Classification) | Predicting movie genres based on plot summaries using NLP. |
| **Task 2** | [Credit Card Fraud Detection](./Task_2_Credit_Card_Fraud_Detection) | Identifying fraudulent transactions using classification algorithms. |
| **Task 3** | [Customer Churn Prediction](./Task_3_Customer_Churn_Prediction) | Predicting customer retention based on demographic and account data. |
| **Task 4** | [Spam SMS Detection](./Task_4_Spam_SMS_Detection) | Classifying messages as spam or ham using Naive Bayes. |
| **Task 5** | [Handwritten Text Generation](./Task_5_Handwritten_Text_Generation) | Generating text patterns using a Character-level RNN. |

## 🛠️ Tech Stack

- **Libraries:** Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
- **Tools:** VS Code, GitHub
